# Awesome-Claude-Code-Source-Code

[English](README.md) | **中文**

精选的 Claude Code 源码研究、逆向工程与自定义项目列表。


## 项目

### 反编译 & 逆向工程
[claude-code](https://github.com/instructkr/claude-code) - 由 @instructkr 发起的 Claude Code 重写项目  
[claude-code](https://github.com/claude-code-best/claude-code) - 由 @claude-code-best 发起的 Claude Code 反编译与逆向工程项目

### 自定义项目
[open-agent-sdk](https://github.com/shipany-ai/open-agent-sdk) - 由 @shipany-ai 开发的开源 Agent SDK，灵感来源于 claude-agent-sdk  
[cc-mini](https://github.com/e10nMa2k/cc-mini) - 由 @e10nMa2k 开发的轻量级 Python 版 Claude Code 社区实现

### 研究

[claude-code-deep-dive](https://github.com/tvytlx/claude-code-deep-dive) - 由 @tvytlx 发起的系统性研究，基于 claude-code npm 包中 cli.js.map 的源码恢复  
[Claude Code Unpacked](https://ccunpacked.dev/) - Agent 循环、50+ 工具、多 Agent 编排以及未发布特性，直接从源码映射而来  
[Harness Books](https://github.com/wquguru/harness-books) - 关于 Harness 工程的两本书：《Harness Engineering：Claude Code 设计指南》和《Claude Code 和 Codex 的 Harness 设计哲学》
